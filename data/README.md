## Dataset Source

The dataset used in this project is the *Olist Brazilian E-Commerce Dataset*, publicly available on Kaggle.

Due to GitHub file size limitations, raw CSV files are not stored in this repository.

You can download the dataset here:
https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce

Tables used:
- olist_orders_dataset.csv
- olist_customers_dataset.csv
- olist_order_items_dataset.csv
- olist_products_dataset.csv
